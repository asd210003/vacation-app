Application Title: Vacation App

Application Purpose: This app is designed to allow users to create,  modify, and delete vacation plans. Within these vacation plans, the
user can have excursions planned out as well. This app allows for as many vacations as the user desires, and will also allow the user 
to have alerts and share their vacation plans.

Directions:
The mobile app will open on the home page with two options:

NEW -
This option allows you to create a new vacation. Here, you can define the parameters for the vacation, but you cannot create
excursions, alarms, or share until you've already saved it. Define the necessary name, hotel, start date and end date. Then, save
the vacation.

VIEW VACATIONS -
This will be blank until you have saved a vacation. From here, you can click on saved vacations and modify their data or delete the vacation. Once you have saved
the vacation, you can also add the start and end alarms, share the vacation, and create excursions.

    ALARMS - These will create an alarm at a predetermined time, 9:00 AM.

    EXCURSION - This will prompt you to a page similar to the vacation definition page. Here, you can set a title and a date for a set excursion
    on your vacation. You can also set an alarm for the excursion and save it. Once the excursion is saved, it will take you back to the page 
    with vacation details, where you can further modify. NOTE: You cannot delete a vacation that has an excursion in it.
